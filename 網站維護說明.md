## 六、產生「PPT ＋ 備忘稿 ＋ 最新資料夾」（交付用）

`...\1150531 PCNE概論\最新\` 放最新成品：

| 檔案 | 說明 |
|---|---|
| `PCNE-DRP概論_講義_最新.pdf` | = handout.pdf（67 頁） |
| `PCNE-DRP概論_簡報_含備忘稿_最新.pptx` | 67 張，每張一頁簡報圖 ＋ 備忘稿在「備註」欄（16:9） |
| `PCNE-DRP概論_備忘稿_最新.md` | 67 頁逐頁備忘稿純文字 |

做法（Python，需 `pip install pymupdf python-pptx`）：
1. 解析 `slides.html` 的 67 個 `<section>` 取每頁 `<aside class="notes">` 當備忘稿（與 PDF 67 頁逐頁對齊）。
2. pymupdf 把 handout.pdf 每頁算成 PNG（dpi 144）。
3. python-pptx 建 16:9 簡報：每頁貼整頁圖、`notes_slide` 寫備忘稿。
4. 複製 handout.pdf、輸出備忘稿 md。

> ⚠ PPTX 是「圖片版」（投影片文字不可在 PPT 內編輯，但備忘稿可編輯）。若要文字可編輯版，需 Marp `--pptx-editable`＋LibreOffice。

## 七、維護注意事項（踩過的坑）

- **OneDrive 上 `git commit` 會壞**：`cannot update the ref 'HEAD'…`。已設定好修法：
  ```powershell
  git config windows.appendAtomically false
  git config core.fsmonitor false
  ```
- `build_slides.py` 需要 `slides_template.html`，**目前缺檔**，所以改簡報請直接改 `slides.html`（並同步改 `pcne-slides.md` / `pcne-marp.md`）。
- OneDrive 雲端檔讀取偶爾卡／回傳亂內容；改檔、git 建議用 Python 腳本跑完再抓線上頁面確認。
- **畫筆（chalkboard）破圖**：簡報右上「畫筆」用的是 reveal.js chalkboard 外掛（`assets/chalkboard/plugin.js` + `style.css`），但**沒有 copy 原版的 `assets/chalkboard/img/` 圖檔資料夾**。plugin.js 寫死要抓 `img/sponge.png`（橡皮擦按鈕 `<img>`）、`img/boardmarker-*.png` / `img/chalk-*.png`（筆游標）、`img/blackboard.png`（黑板底圖）。一點畫筆，橡皮擦那個 `<img>` 載不到 → 左上角跳「破圖 icon」（游標和 CSS 底圖載不到只會靜默 fallback，不會跳破圖）。
  - **解法（不依賴外部圖檔）**：在 `slides.html` 的 `chalkboard:{}` 設定覆寫 → `eraser.src` 用內嵌 SVG、7 色 `boardmarkers`＋7 色 `chalks` 游標改 CSS `crosshair`、黑板 `background` 用內嵌深綠 SVG。這些 key 在 plugin.js 的 `configure()` 都有 `if(config.x)` 可覆寫。
  - **想改筆色／網格／黑板底色**：都在 `slides.html` 同一個 `chalkboard:{}` 區塊——`boardmarkers[]`（投影片上畫筆，預設第 1 個紅色）、`chalks[]`（黑板粉筆，預設第 1 個亮黃）、`background[1]`（黑板底色 SVG）、`grid:false`（網格）、`chalkEffect`（>0；噴點程度，越小越像實線，0 會被 plugin 忽略）。陣列第 1 個顏色＝預設色。
  - 若哪天想恢復原版筆刷圖示／黑板貼圖，改成把原版 reveal.js-chalkboard 的 `img/` 整個資料夾放進 `assets/chalkboard/img/` 即可（兩種修法擇一，不要並用）。
